######
xm_rst
######


Usage
#####

.. code:: bash

   xm_rst --help


Installation
############


Requirements:

- docutils
- argcomplete


Simply:

.. code:: bash

   mkdir -p ~/.local/opt
   pushd ~/.local/opt
   git clone ...
   popd


And you can add to your environment:

.. code:: bash

   eval "$(register-python-argcomplete xm_rst 2>/dev/null)"
